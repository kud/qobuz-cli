**qobuz-cli** brings Qobuz to your terminal: search the catalogue, manage
favourites and playlists, open anything in the desktop app, and drive playback —
without leaving the shell. It's a thin surface over the
[`@kud/qobuz`](/projects/qobuz) core: the library does the work, the CLI formats
the output.

## 🎛️ How it fits together

Two halves work in tandem:

- **The API side** — search, metadata, and library/playlist management talk to
  Qobuz through the core library, authenticated by a browser-borrowed token in
  your macOS Keychain.
- **The playback side** — `play`, `next`, `previous`, and friends drive the
  **Qobuz desktop app** via real media keys (macOS, needs Accessibility
  permission), and `open` / `url` deep-link into it. The CLI steers the app you
  already have; it doesn't stream audio itself.

## 📦 Install

```sh
npm install -g @kud/qobuz-cli
```
